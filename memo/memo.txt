# 기본적인 개념 정리 용도.
# 이후에 삭제 예정


하나의 웹사이트를 위해 html, css, js 3개의 파일을 사용
각각의 역할은 다음과 같다.

html : 웹페이지의 기본 구조를 구성.
css  : 웹페이지의 모양이나 테마를 구성.
js   : 웹페이지의 동작이나 기능을 구성.


html의 구조는 다음과 같다.
기본 뼈대
    !DOCTYPE: 문서 유형 정의.
    <html>  : 문서 전체를 감싸는 최상위 태그
head
    title   : 제목 구성
    css와의 파일 연결
body
    <h1>    : 기본 텍스트
    <input> : 입력
    <button>: 버튼
