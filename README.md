# web-peoject
html, css, js를 이용한 웹사이트 제작 연습용 프로젝트입니다.
웹페이지: [text](https://ocean1198.github.io/web-project/)

# index.html
각 홈페이지로의 이동을 지원합니다.
현재는 02_avoid의 개발을 위해 해당 페이지로만 리다이렉트됩니다.

# 01_basic
html의 기본 구성과 js와의 연결을 다룹니다.
html의 Button을 js에서 불러와 기능을 실행합니다.

# 02_avoid
html의 canvas 구성을 이용해 간단한 피하기 게임을 제작했습니다.
canvas의 작동, html-JS 간의 변수 공유, CSS의 기초 스타일에 대한 내용이 포함됩니다.

# 03_sudoku
sudoku의 문제를 만드는 페이지입니다.
현재는 python을 이용해 sudoku에 관련된 알고리즘을 학습하고 있으며, 이후에 추가할 예정입니다.
